# 09 · 设计系统

> **状态说明（2026-07 v2 重构后）**：本文档记录的是 v1 设计系统（TDesign 蓝 #0052D9）。现行 UI 已按 v2 重设计实施——品牌色取 logo 主色 `#0053DB`、小圆角 5/8/10/12、卡片化布局、毛玻璃侧栏，高保真原型见 `design/`（`index.html` 为入口），Compose 侧实现以 `src/**/ui/theme/`（`DesignTokens.kt`、`Theme.kt`）为准。以下 v1 内容保留作历史对照。

> 源自 `ai-context/design-rules.md` + `app/styles/tokens.css`。
> 视觉学派：**Modern Minimal**（Tech Utility）· **TDesign Enterprise × macOS Native**。
> CMP 重构需完整复刻这套 design token 与组件视觉。

---

## 1. 设计哲学

- **Philosophy**：功能第一，克制美学——像 macOS 原生应用一样精准、高效、无冗余装饰
- **Direction**：`minimal, utilitarian, information-dense, mac-native`
- **Personality**：专业、克制、高效、可信赖
- **Reference**：TDesign Desktop + macOS Finder

## 7. 组件样式规范

### Button

| 类型 | 样式 |
|---|---|
| 主按钮 | `background: brand; color: #fff; border-radius: 3px; height: 36px`(登录 40px) |
| 文字按钮 | `background: transparent; color: brand` hover → `brand-lighter` |
| 图标按钮 | `32×32; border-radius: 3px; color: text-secondary` hover → `bg-hover, text-primary` |

### Input

`height: 32px; border: 1px solid border; border-radius: 3px` focus → `border-color: brand; background: #fff`

### Status Tag

`padding: 1px 6px; border-radius: 3px; font-size: 12px`，四种：`tag-primary` / `tag-success` / `tag-warning` / `tag-error`

### File List Row

`grid: 1fr 100px 140px 110px; height: 44px; border-bottom: 0.5px solid border` hover → `bg-hover`；active → `brand-lighter`

## 8. SVG 图标系统（32 个）

所有图标用内联 SVG `<symbol>` + `<use>` 引用，无外部图标库。源码：`app/components/IconSprite.vue`。

| ID | 用途 |
|---|---|
| `i-cloud` | 云朵 · 品牌 Logo |
| `i-local` | 本地（已同步到本地状态） |

| `i-file` / `i-file-text` | 通用文件 / 文本文档 |
| `i-image` / `i-chart` / `i-video` | 图片 / 图表 / 视频 |
| `i-search` / `i-refresh` | 搜索 / 刷新 |
| `i-transfer` / `i-settings` | 传输 / 设置 |
| `i-check` / `i-sync` / `i-alert` / `i-clock` | 已完成 / 同步中(可旋转) / 冲突 / 等待 |
| `i-copy` / `i-info` / `i-arrow` | 复制 / 属性 / 箭头 |

## 10. 禁止模式（Never Do）

- ❌ 不要圆形按钮（一律 3px / 6px 圆角）
- ❌ 不要 ripple / 水波纹动画（macOS 原生无此交互）
- ❌ 不要鲜艳大渐变背景（仅登录页允许浅蓝→白渐变）
- ❌ 不要 emoji 作功能图标（用 SVG）
- ❌ 不要超过 14px 的正文字号
- ❌ 不要多层卡片嵌套阴影（同一界面只有一层 shadow）
- ❌ 不要底部导航栏
- ❌ 不要外部 CDN 依赖

### 推荐（Prefer）

- ✅ 用 `border-bottom: 0.5px solid border` 做分隔
- ✅ 悬停用 `background: bg-hover` 而非颜色深化
- ✅ 状态标识用 `<span class="tag tag-success">`
- ✅ 动画提供 reduced-motion 降级

## 11. 页面规格

### 登录页
- macOS 窗口（1280×800）
- 渐变背景 + 3 个低透明度装饰圆（0.04-0.06 opacity）
- 居中白色卡片（`max-width: 480px`, `border-radius: 9px`）
- 品牌区：64px 蓝底圆角图标 + 标题 + 蓝色分隔线
- 主按钮「使用华为账号登录」（全宽，40px 高，带云朵图标）

### 主界面（最核心）
- 双栏：左侧栏 220px（Logo + 目录树 + 账号）+ 右侧主区
- 右侧：AppBar(56px) + 面包屑(32px) + 同步状态条(32px) + 文件列表
- 文件列表 **6 列**：checkbox(40px) / 名称(flex) / 大小(100px,可拖拽) / 修改时间(150px,可拖拽) / 同步状态(60px) / 操作(40px)
- 传输 Popover(420×560)：点击「传输」按钮弹出，**无 Tab 切换**——全部任务单列表展示，靠 `stateMeta` 映射每状态的 icon/label/color

### 设置页
- 双栏：左侧导航 200px（同步目录/传输设置/高级设置/日志查看/关于）+ 右侧设置区
- 控件：Stepper 步进器、数字输入、Toggle 开关
- 底部：保存设置 + 重置默认 + 自动保存状态
